## 📱 Descripción

**AlkeWallet** es una aplicación móvil de billetera digital desarrollada en **Kotlin** para Android. Permite a los usuarios gestionar su saldo de forma sencilla e intuitiva, con funciones para registrar información personal y realizar operaciones de depósito y retiro de dinero.

---

## ✨ Funcionalidades

| Función | Descripción |
|---|---|
| 📋 **Formulario de registro** | Completa tus datos personales para configurar tu cuenta |
| ➕ **Sumar saldo** | Agrega dinero a tu billetera digital |
| ➖ **Restar saldo** | Retira o descuenta dinero de tu saldo disponible |
| 💰 **Consulta de saldo** | Visualiza tu saldo actual en tiempo real |

## 🚀 ¿Cómo ejecutar el proyecto?

### Requisitos previos
- Android Studio instalado
- SDK de Android configurado
- Dispositivo físico o emulador Android

### Pasos

**1.** Clona el repositorio:
```bash
git clone https://github.com/DeniBeatriz/AlkeWallet_App.git
```

**2.** Abre el proyecto en Android Studio:
```
File → Open → Selecciona la carpeta del proyecto
```

**3.** Espera a que Gradle sincronice las dependencias

**4.** Ejecuta la app:
```
Run → Run 'app'  o  Shift + F10
```
